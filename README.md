
# Repositório de Exercícios e Projetos em C++

Este repositório contém uma série de pastas relacionadas aos estudos e projetos realizados no âmbito da linguagem de programação C++. Aqui está uma breve descrição de cada pasta:

## C++ - Lógica da Programação
Esta pasta contém exercícios e exemplos relacionados à lógica de programação utilizando a linguagem C++. Os exercícios podem variar em complexidade, desde conceitos básicos até desafios mais avançados.

## EAD - C++ LDP (Ensino à Distância - C++ Lógica de Programação)
Nesta pasta, você encontrará materiais e atividades relacionados a cursos ou estudos de lógica de programação com a linguagem C++ realizados à distância (EAD). Os recursos podem incluir slides, exercícios, e outros materiais de apoio.

## Projetos Pessoais - LDP (Lógica de Programação)
Aqui estão projetos pessoais que foram desenvolvidos utilizando a linguagem C++ como base. Esses projetos podem abranger desde pequenas aplicações até programas mais complexos, todos focados no desenvolvimento e aplicação de conceitos de lógica de programação.

## SA - Sistema de Controle de Vendas em C++
Esta pasta contém um projeto específico de um Sistema de Controle de Vendas implementado em C++. O projeto pode incluir vários arquivos fonte, documentação e outros recursos relacionados ao desenvolvimento e funcionamento do sistema.

## LP - Lógica de Programação
Nesta pasta, você encontrará materiais gerais relacionados ao estudo de lógica de programação, não necessariamente limitados à linguagem C++. Isso pode incluir exercícios, teoria, e outros recursos úteis para aprimorar suas habilidades de programação.
